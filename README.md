# MMOKitchen

## Description

4 Staff just isn't enough? Hire 8 more for your team!

MMOKitchen allows up to 12 more players to join your game!

## Requirements

- KitchenLib >= 0.3.0

## Installation

- Subscribe on the Steam Workshop
